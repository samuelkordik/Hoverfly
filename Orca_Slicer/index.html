2 · Machine limits (match firmware)

In Printer settings → Motion ability, mirror the firmware so Orca never asks for more than the board allows.

LimitValueFirmware source
Max speed X / Y500 / 500 mm/sDEFAULT_MAX_FEEDRATE
Max speed Z / E5 / 25 mm/sDEFAULT_MAX_FEEDRATE
Max accel X / Y500 / 500 mm/s²DEFAULT_MAX_ACCELERATION — see warning
Max accel E5000 mm/s²DEFAULT_MAX_ACCELERATION
Jerk X / Y8 / 8 (or leave low)Classic jerk is OFF; firmware uses Junction Deviation 0.08, so Orca jerk is mostly advisory
Speed bottleneck: firmware max acceleration is a conservative 500 mm/s². No matter how high you set slicer speeds, the printer can't accelerate to them on short moves — so a "fast" profile is largely wasted until you raise firmware accel. Do Calibration Step 7 first and lift firmware accel to ~1500–2000 mm/s² (then raise Orca's max accel to match). The Speed profile below assumes you've done this.
Retraction speed is capped at 25 mm/s. The E max feedrate above is 25 mm/s, so Orca silently clamps any retraction speed you set higher than that. Set retraction speed to 25 mm/s (not the 35–45 you'll see quoted for Bowden setups) — or raise the firmware E feedrate (M203 E50 + M500) if you really want faster retracts. A light Sprite direct drive rarely needs more.

The Sprite is light and PLA cools fast, so the Speed numbers are realistic once firmware accel is raised. Keep outer-wall speed lower than inner even on Speed — the outer surface is what you see.

5 · Start / end G-code

Start G-code that homes, meshes-load, heats, and (optionally) does an auto bed level:

; ---- START ----
M104 S[nozzle_temperature_initial_layer] ; start warming hotend
M140 S[bed_temperature_initial_layer]    ; bed
G28                                       ; home all
M420 S1 Z10                               ; load saved mesh + fade height (skip G29 to save time)
M190 S[bed_temperature_initial_layer]     ; wait for bed
M109 S[nozzle_temperature_initial_layer]  ; wait for hotend
G92 E0
G1 Z2.0 F3000
G1 X2 Y20 Z0.3 F5000                       ; prime line
G1 X2 Y200 E15 F1500
G92 E0
; ---- END ----
M104 S0
M140 S0
G91
G1 E-2 F2700
G1 Z10 F600
G90
G1 X5 Y200 F3000   ; present the print
M84                ; steppers off
The start G-code uses M420 S1 to load the stored mesh instead of re-probing every print. Run G29 + M500 once (Calibration Step 9) so a good mesh exists. Add G29 back to the start G-code if you'd rather probe every time.

6 · Orca's built-in calibration order

Orca has a Calibration menu — run these in order, then save the results into your filament/process presets. Each maps to a step in the Calibration Guide:

  1. Temperature tower → set nozzle temp (Cal §5)
  2. Flow rate → set flow ratio (Cal §3)
  3. Pressure advance → set PA, or use Marlin M900 K instead (Cal §6)
  4. Retraction test → set retraction distance/speed (Cal §6)
  5. Max volumetric speed → set the PLA throughput cap (Cal §7)
